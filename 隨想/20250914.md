# <隨想>  

你越渴望財富，那麼財富會離你遠去  
你越渴望愛情，那麼愛情會離你遠去  
你越渴望知識，那麼知識會離你遠去  
你越渴望成為智者，那你便很難成為智者


# <隨筆> 
深夜，在榕樹下偶遇了祂  
在祂離開前，我著急忙荒得問道:「為什麼是我?」  
祂轉頭看向我:「為什麼不能是你?」  
我想了想，小聲說:「我既不聰明，又沒天賦，  
記憶力差，老是把事情搞砸，從小到大跌跌撞撞，  
我感覺這系統就沒啥人喜歡我」  

祂聳聳肩:「沒錯，從小到大你就沒做過啥正確的選擇，  
說腦袋不好，也確實是不好，而且，你就是個自私的混蛋」  
我不置可否的微微點頭，祂轉頭邁步離去  

「也許是你本質的善良吧」祂說:「還有包容他人的能力」  
「你」祂漸漸消失在前方的馬路口:「沒自己想得差」
